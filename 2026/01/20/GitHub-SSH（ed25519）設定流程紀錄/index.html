Windows 上打開 Git Bash
Windows 上打開 Git Bash第一步：開啟命令提示字元（cmd） 鍵盤按下 1Windows 鍵 + R 輸入： 1cmd 按 Enter 看到畫面類似： 1C:\Users\username> 代表已成功開啟。 第二步：確認 Git 是否已安裝在 cmd 中輸入： 1git --version 正常會看到類似結果：1git version 2.51.2.windows.1 能看到版本號即可，這代表 Git 已安裝完成(版本號可能不同) 第三步：打開 Git Bash 鍵盤按下 1Windows 鍵 輸入： 1Git Bash 按 Enter 補充說明 git 可以在 cmd 使用 git-bash.exe 預設不在系統指令清單中， 因此無法在 cmd 小黑窗中直接輸入 git-bash 來開啟 這是 Git for Windows 的預設設計，不是安裝錯誤 結論用開始功能表打開 Git Bash，是最穩定做法
