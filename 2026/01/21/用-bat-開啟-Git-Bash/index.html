用 bat 開啟 Git Bash

有時需要在指定資料夾快速開啟 Git Bash。
在 Windows 下，可以透過一個簡單的 .bat 檔完成這件事。

目標行為

執行 .bat 後：

  • 開啟一個新的 Git Bash 視窗
  • 目錄自動切換到 .bat 所在的資料夾
  • Bash 保持互動狀態，可直接輸入指令

bat 內容

行為說明

  • %~dp0
    代表目前這個 .bat 檔案所在的資料夾路徑

  • start "" ...
    以新視窗啟動程式，避免 bat 結束時影響 Git Bash

  • bash.exe --login -i
    以 login 且互動模式啟動 bash

  • cd '目錄'; exec bash
    切換目錄後，啟動新的互動 shell，使視窗持續存在

執行後的狀態

成功啟動後，Git Bash 會顯示類似以下提示符：

這表示：

  • Bash 正在互動模式中執行
  • 目前目錄已切換完成
  • Shell 正在等待輸入指令

使用方式

  1. 將上述內容存成 .bat 檔案
  2. 放入任意資料夾
  3. 雙擊執行

Git Bash 會在該資料夾中開啟。

補充

  • 若 Git 安裝在不同路徑，需調整 bash.exe 的位置
  • 此寫法不依賴右鍵選單或系統整合
  • 僅作為快速開啟 Git Bash 的工具使用

結尾

這是一個單純用來啟動 Git Bash 的 bat。
行為固定、結構簡單，適合放在需要的資料夾中直接使用。
